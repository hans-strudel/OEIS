%N A048098 Sqrt(n)-smooth numbers: if p | n then p^2 <= n when p is prime.
%C A048098 This set (say S) has density d(S) = 1-Log(2) and multiplicative density m(S) = 1-exp(-Gamma). Multiplicative density : let A be a set of numbers, A(x) = { k in A | gpf(k) <=x } where gpf(k) denotes the greatest prime factor of k and let m(x)(A) = prod(p<=x, (1-1/p))*sum(k in A(x), 1/k). If lim x ->infinity m(x)(A) exists = m(A), this limit is called "multiplicative density" of A (Erdős and Davenport, 1951). - _Benoit Cloitre_, Jun 12 2002
