%N A062366 Let f be a function on rationals p/q (p,q coprime) defined by f(p/q) = abs(p-q)/nextoddColl(p) (so the denominator is the next odd number in the Collatz trajectory of p). Start with f(n/1) and repeat application of f until it reaches again an integer, which is a(n). If no integer is reached, then a(n)=0.
%C A062366 Is a(n)<=n for all n?
%e A062366 For n=15 we get 15/1 -> 14/23 -> 9/7 -> 2/7 -> 5/1 so a(15)=5.
