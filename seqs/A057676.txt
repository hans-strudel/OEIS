%N A057676 Smallest prime q(n) such that -q(n)+2^p(n) is a prime, where p(n) = n-th prime.
%e A057676 n = 4, p(4) = 11, 2^11 = 2048, p2 = 2048-p1 is satisfied at first with prime p1 = 19 resulting in prime p2 = 2029, so q(4) = 19. n = 31, p(31) = 127, p2 = 2^127-p1 is satisfied first with p1 = 577 and p2 = 170141183460469231731687303715884105151 So q(31) = 577, the 106th prime.
