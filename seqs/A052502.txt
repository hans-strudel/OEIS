%N A052502 Number of permutations sigma of [3n] without fixed points such that sigma^3 = Id.
%C A052502 For n >= 1 a(n) is the size of the conjugacy class in the symmetric group S_(3n) consisting of permutations whose cycle decomposition is a product of n disjoint 3-cycles.
%H A052502 INRIA Algorithms Project, <a href="http://algo.inria.fr/ecs/ecs?searchType=1&amp;service=Search&amp;searchTerms=27">Encyclopedia of Combinatorial Structures 27</a>
%F A052502 a(n) = (3*n)!/(n!*3^n). Using Stirling's formula in A000142 we have a(n) ~ sqrt(3) * 9^n * (n/e)^(2n). - Ahmed Fares (ahmedfares(AT)my-deja.com), Apr 21 2001
%F A052502 E.g.f.: (every third coefficient of) exp((x^3)/3).
%F A052502 G.f.: hypergeom([1/3, 2/3, 1], [], 9*x).
%F A052502 Recurrence: a(0) = 1, a(n) = (3*n-1)*(3*n-2)*a(n-1) for n >= 1.
%F A052502 Write the generating function for this sequence in the form A(x) = sum {n >= 0} a(n)* x^(2*n+1)/(2*n+1)!. The g.f. A(x) satisfies A'(x)*( 1 - A(x)^2) = 1. _Robert Israel_ remarks that consequently A(x) is a root of z^3 - 3*z + 3*x with A(0) = 0. Cf. A001147, A052504 and A060706. - _Peter Bala_, Jan 02 2015
