%C A006784 Definition of Pierce expansion: for a real number x (0<x<1), there is always a unique increasing positive integer sequence (a(i))_i>0 such that x = 1/a(1) - 1/a(1)/a(2) + 1/a(1)/a(2)/a(3) -1/a(1)/a(2)/a(3)/a(4) .. This expansion can be computed as follows: let u(0)=x and u(k+1) = u(k)/(u(k)-floor(u(k)); then a(n)=floor(u(n)). - _Benoit Cloitre_, Mar 14 2004

%F A006784 Definition of Engel expansion: For a positive real number x (here Pi), define 1 <= a(1) <= a(2) <= a(3) <= ... so that x = 1/a(1) + 1/a(1)a(2) + 1/a(1)a(2)a(3) + ... by x(1)=x, a(n) = ceiling(1/x(n)), x(n+1) = x(n)a(n)-1. Expansion always exists and is unique. See references for more information.
