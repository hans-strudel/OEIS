%N A005204 Coding a recurrence.
%C A005204 Consider a rabbits generation tree, and code each level with 0 for a single segment, and 1 for a branched segment. The current sequence written in binary: 0, 0, 0, 1, 10, 100, is obtained with this scheme applied on sequence A000930, and follows recurrence formula a(n+3) = 2^A000930(n-1)*a(n+2) + a(n), when n>=3. Note that Fib. Quart. article gives incorrect value of 158022 for a(10). - _Michel Marcus_, Jul 29 2013
%D A005204 N. J. A. Sloane and Simon Plouffe, The Encyclopedia of Integer Sequences, Academic Press, 1995 (includes this sequence).
