%N A002117 Decimal expansion of zeta(3) = sum(m>=1, 1/m^3).
%C A002117 Sometimes called Apery's constant.
%C A002117 "A natural question is whether Zeta(3) is a rational multiple of Pi^3. This is not known, though in 1978 R. Apery succeeded in proving that Zeta(3) is irrational. In Chapter 8 we pointed out that the probability that two random integers are relatively prime is 6/Pi^2, which is 1/Zeta(2). This generalizes to: The probability that k random integers are relatively prime is 1/Zeta(k) ... ." [Stan Wagon]
%C A002117 In 2001 Tanguy Rivoal showed that there are infinitely many odd (positive) integers at which zeta is irrational, including at least one value j in the range 5 <= j <= 21 (refined the same year by Zudilin to 5 <= j <= 11), at which zeta(j) is irrational. See the Rivoal link for further information and references.
%C A002117 The reciprocal of this constant is the probability that three integers chosen randomly using uniform distribution are relatively prime. - Joseph Biberstine (jrbibers(AT)indiana.edu), Apr 13 2005
%C A002117 Also the value of zeta(1,2), the double zeta-function of arguments 1 and 2. - _R. J. Mathar_, Oct 10 2011

%F A002117 From _Peter Bala_, Dec 04 2013: (Start)
%F A002117 zeta(3) = 16/7*sum {k even} (k^3 + k^5)/(k^2 - 1)^4.
%F A002117 zeta(3) - 1 = sum {k >= 1} 1/(k^3 + 4*k^7) = 1/(5 - 1^6/(21 - 2^6/(55 - 3^6/(119 -...- (n - 1)^6/((2*n - 1)*(n^2 - n + 5) - ...))))) (continued fraction).
%F A002117 More generally, there is a sequence of polynomials P(n,x) (of degree 2*n) such that
%F A002117 zeta(3) - sum {k = 1..n} 1/k^3 = sum {k >= 1} 1/( k^3*P(n,k-1)*P(n,k) ) = 1/((2*n^2 + 2*n + 1) - 1^6/(3*(2*n^2 + 2*n + 3) - 2^6/(5*(2*n^2 + 2*n + 7) - 3^6/(7*(2*n^2 + 2*n + 13) - ...)))) (continued fraction). See A143003 and A143007 for details.
%F A002117 Series acceleration formulas:
%F A002117 zeta(3) = 5/2*sum {n >= 1} (-1)^(n+1)/( n^3*binomial(2*n,n) )
%F A002117 = 5/2*sum {n >= 1} P(n)/( (2*n(2*n - 1))^3*binomial(4*n,2*n) )
%F A002117 = 5/2*sum {n >= 1} (-1)^(n+1)*Q(n)/( (3*n(3*n - 1)*(3*n - 2))^3*binomial(6*n,3*n) ), where P(n) = 24*n^3 + 4*n^2 - 6*n + 1 and Q(n) = 9477*n^6 - 11421*n^5 + 5265*n^4 - 1701*n^3 + 558*n^2 - 108*n + 8 (Bala, section 7). (End)
%F A002117 zeta(3) = Sum_{n >= 1} (A010052(n)/n^(3/2)) = Sum_{n >= 1} ( (floor(sqrt(n)) - floor(sqrt(n-1)))/n^(3/2) ). - _Mikael Aaltonen_, Feb 22 2015
