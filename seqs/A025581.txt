%C A025581 A problem posed by François Viète (Vieta) in his book Zeteticorum liber quinque (1593), liber 2, problem 19, (quoted in the Alten et al. reference, on p. 292) is to find for a rectangle (a >= b >= 1) with given a^3 - b^3, name it C, and a*b, name it F, the difference a-b, name it x. This is a simple exercise which Viète found remarkable. It reduces to a standard cubic equation for x, namely  x^3 + 3*F*x = C. Proof: Use the square of the diagonal d^2 = a^2 + b^2. Then (i) C = a^3 - b^3 = (a - b)*(a^2 + b^2 + a*b) = x*(d^2 + F). (ii) use the trivial relation d^2 = (a-b)^2 + 2*a*b = x^2 + 2*F, to eliminate d^2 in (i). End of the Proof. Here for positive integers a = n and b = k: (T(n, k)^2 + 3*A079904(n, k))*T(n, k) = A257238(n, k) (also true for n = k = 0). - _Wolfdieter Lang_, May 12 2015
