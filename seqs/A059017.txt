%N A059017 Smallest n-digit prime whose square is a concatenation of two n-digit primes (no leading zeros allowed), 0 if no such prime.
%e A059017 E.g. 5449^2 = 29691601 = (2969)(1601).
%o A059017 (PARI) { a(n) = local(t,m,s); t = 10^(n-1); while(1, t=nextprime(t+1); m = nextprime(sqrtint(t*10^n)+1); s=m^2; if( s\10^n!=t, next); if( (s%10^n) > 10^(n-1) && ispseudoprime(s%10^n), return(m); ); ) } [From _Max Alekseyev_, Apr 23 2010]
