%N A004490 Colossally abundant numbers: n for which there is a positive exponent epsilon such that sigma(n)/n^{1 + epsilon} >= sigma(k)/k^{1 + epsilon} for all k > 1, so that n attains the maximum value of sigma(n)/n^{1 + epsilon}.
