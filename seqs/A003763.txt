%N A003763 Number of Hamiltonian cycles on 2n X 2n square grid of points.
%C A003763 Orientation of the path is not important; you can start going either clockwise or counterclockwise.
%C A003763 The number is zero for a 2n+1 X 2n+1 grid (but see A222200).
%C A003763 These are also called "closed rook tours".
