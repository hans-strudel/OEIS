%N A005864 The coding-theoretic function A(n,4).
%C A005864 Since A(n,3) = A(n+1,4), A(n,3) gives essentially the same sequence.
%C A005864 The next term is in the range 2816-3276.
%C A005864 Let T_n be the set of SDS-maps of sequential dynamical systems defined over the complete graph K_n in which all vertices have the same vertex function (defined using a set of two possible vertex states). Then a(n) is the maximum number of period-2 orbits that a function in T_n can have. - _Colin Defant_, Sep 15 2015
%C A005864 a(n) is identical to the independence number of the n-halved cube graph up to at least n = 12. It seems reasonable to conjecture the two are identical for all n. - _Eric W. Weisstein_, Dec 31 2015
