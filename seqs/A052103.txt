%N A052103 The third of the three sequences associated with the polynomial x^3 - 2.
%C A052103 If x^3 = 2 and n >= 0, then there are unique integers a, b, c such that (1 + x)^n = a + b*x + c*x^2. The coefficient c is a(n).
%D A052103 R. Schoof, Catalan's Conjecture, Springer-Verlag, 2008, pp. 17-18.
%H A052103 A. Kumar Gupta, A. Kumar Mittal, <a href="http://arXiv.org/abs/math.GM/0001112">Integer Sequences associated with Integer Monic Polynomial</a>, arXiv:math.GM/0001112, Jan 2000.
%H A052103 <a href="/index/Rec#order_03">Index entries for linear recurrences with constant coefficients</a>, signature (3, -3, 3).
%F A052103 a(n) = 3 * a(n-1) - 3 * a(n-2) + 3 * a(n-3), n >2.
%F A052103 Sum[0..[n/3], 2^k * C(n, 3k+2)]. - _Ralf Stephan_, Aug 30 2004
%F A052103 We have: a(n)=4a(n-1)-6a(n-2)+6a(n-3)-3a(n-4). a(n) is binomial transform of 0, 0, 1, 0, 0, 2, 0, 0, 4, 0, 0, 8, 0, 0, 16, 0, 0, 32 (see A077958). Also a(n) is a sequence identical to half its third differences. - _Paul Curtz_, Mar 10 2008
%F A052103 O.g.f.: -x^2/(-1+3*x-3*x^2+3*x^3). a(n+1)-a(n)=A052102(n). - _R. J. Mathar_, Apr 01 2008
%e A052103 x^2 + 3*x^3 + 6*x^4 + 12*x^5 + 27*x^6 + 63*x^7 + 144*x^8 + ...
