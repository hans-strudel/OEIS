%N A051224 Number of ways of placing n nonattacking superqueens on n X n board (symmetric solutions count only once).
%C A051224 A superqueen moves like a queen and a knight.
%H A051224 D. Bill, <a href="http://www.durangobill.com/N_Queens.html">Durango Bill's The N-Queens Problem</a>
%H A051224 W. Schubert, <a href="http://m29s20.vlinux.de/~wschub/nqueen.html">N-Queens page</a> - _W. Schubert_, Nov 29 2009
%F A051224 a(n) = (1/8) * (Q(n) + P(n) + 2 * R(n)), where Q(n) = A051223(n) [all solutions], P(n) [point symmetric solutions (180 degrees)] and R(n) [rotationally symmetric solutions (90 degrees)]. This formula has the same structure as the formula for A002562. There seem to be no OEIS sequences (yet) for P(n) and R(n). See http://m29s20.vlinux.de/~wschub/nqueen.html for a few terms. - _W. Schubert_, Nov 29 2009
