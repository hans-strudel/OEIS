%F A000113 Let psi(m) = A001615(m) (Dedekind's psi function). Write n = 2^i*3^j*k, where (6,k) = 1 and let i' = floor(i/2) for i < 6, i' = 3 for i >= 6; let j' = 0 for j = 0 or 1, j' = 1 for j >= 2. Then a(n) = psi(n/(2^i'*3^j')) = psi(n)/(2^i'*3^j').
