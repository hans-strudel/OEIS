%N A004019 a(0) = 0; for n>0, a(n) = (a(n-1) + 1)^2.
%C A004019 Take the standard rooted binary tree of depth n, with 2^(n+1) - 1 labeled nodes. Here is a poor picture of the tree of depth 3:
%C A004019          R
%C A004019        /   \
%C A004019       /     \
%C A004019      o       o
%C A004019     / \     / \
%C A004019    o   o   o   o
%C A004019   / \ / \ / \ / \
%C A004019   o o o o o o o o
%C A004019 Let the number of rooted subtrees be s(n). For example, for n = 1 the s(2) = 4 subtrees are:
%C A004019   R   R   R      R
%C A004019      /     \    / \
%C A004019     o       o  o   o
%C A004019 Then s(n+1) = 1 + 2*s(n) + s(n)^2 = (1+s(n))^2 and so s(n) = a(n+1).
