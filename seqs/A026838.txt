%N A026838 Number of partitions of n into distinct parts, the greatest being even.
%C A026838 Fine's theorem: a(n) - A026837(n) = 1 if n = k(3k+1)/2, = -1 if n = k(3k-1)/2, = 0 otherwise (see A143062).
