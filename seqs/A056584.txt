%N A056584 Solution to (n^2/a(n))^a(n) = GCD[n^n, product(k^k) for k<n] where a(n) and n^2/a(n) are integers, or 0 if no such integers.
%C A056584 For n = 4, there are no integer solutions of (16/a)^a = 4, though there are two real solutions of about 14.5454938 and 0.3673156945.
%F A056584 a(2) = 4, a(4) = 0, a(8) = 4, a(9) = 3; if p an odd prime: a(p) = p^2 and a(2p) = p; otherwise if n>1: a(n) = n. Apart from n = 4, a(n) = n^2/A056583(n) = log(A056582(n))/log(A056583(n)).
