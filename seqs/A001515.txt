%F A001515 The following formulae can all be found in (or are easily derived from formulae in) Grosswald's book.
%F A001515 a(0) = 1, a(1) = 2; thereafter a(n) = (2*n-1)*a(n-1) + a(n-2).
%F A001515 E.g.f.: exp(1-sqrt(1-2*x))/sqrt(1-2*x).
%F A001515 a(n) = Sum_{ k = 0..n } binomial(n+k,2*k)*(2*k)!/(k!*2^k).
%F A001515 Equivalently, a(n) = Sum_{ k = 0..n } (n+k)!/((n-k)!*k!*2^k) = Sum_{ k = n..2n } k!/( (2n-k)!*((k-n)!*2^(k-n)).
%F A001515 Equivalently, as the value of a hypergeometric function: a(n) = 2F0[ n+1, -n ; - ; -1/2].
%F A001515 a(n) ~ exp(1)*(2n)!/(n!*2^n) as n -> oo. [See Grosswald, p. 124]
%F A001515 a(n) = A144301(n+1).
%F A001515 G.f.: 1/(1-x-x/(1-x-2x/(1-x-3x/(1-x-4x/(1-x-5x/(1-.... (continued fraction). - _Paul Barry_, Feb 08 2009
%F A001515 a(-1 - n) = a(n) and (a(n+1) + a(n+2))^2 = a(n)*a(n+2) + a(n+1)*a(n+3) for all integer n. - _Michael Somos_, Apr 08 2012
%F A001515 G.f.: 1/G(0)  where G(k)=  1 - x - x*(2*k+1)/(1 - x - 2*x*(k+1)/G(k+1)); (continued fraction). - _Sergei N. Gladkovskii_, Oct 05 2012
