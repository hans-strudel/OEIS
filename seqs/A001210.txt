%N A001210 a(n) = solution to the postage stamp problem with 5 denominations and n stamps.
%C A001210 _Fred Lunnon_ [W. F. Lunnon] defines "solution" to be the smallest value not obtainable by the best set of stamps. The solutions given are one lower than this, that is, the sequence gives the largest number obtainable without a break using the best set of stamps.
