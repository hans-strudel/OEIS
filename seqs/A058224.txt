%N A058224 Largest d such that the linear programming bound for quantum codes of length n is feasible for some real K>1.
%C A058224 Bounded above by floor((n+1)/6)+floor((n+2)/6)+1 for all n, with equality when n < 100. For n < 22 and 25 <= n <= 30 this bound is attained by actual additive quantum codes; for other values of n, this is unknown.
