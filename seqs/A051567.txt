%N A051567 Consider problem of placing N queens on an n X n board so that each queen attacks precisely k others. Here k=1 and sequence gives number of inequivalent solutions when N is equal to the upper bound 2*floor(2n/3).
%C A051567 a(n) = 0 if N does not achieve 2*floor(2n/3).
