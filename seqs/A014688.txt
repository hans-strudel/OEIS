%C A014688 May be obtained by a sieve on the sequence of natural numbers. Starting from n=1 delete the number corresponding to the alternate sum of the preceding left numbers. Iterate with the successive left number. First step n = 1, k = 1 - 0 = 1: delete the k-th number after n -> 2. Move to successive remaining number n = 3. Then k = 3 - 1 + 0 = 2: delete the k-th number after n -> 5. Move to successive remaining number n = 4. Then k = 4 - 3 + 1 - 0 = 2. After 4 we have 6, 7, 8, ... (5 deleted in previous step). So delete n = 7. And so on. - _Paolo P. Lava_ and _Giorgio Balzarotti_, Jul 14 2008
