%N A001289 Number of equivalence classes of Boolean functions modulo linear functions.
%C A001289 Number of equivalence classes of all 2^(2^n) maps from GF(2)^n to GF(2), where maps f and g are equivalent iff there exists an invertible n X n binary matrix M, two n-dimensional binary vectors a and b and a binary scalar c such that g(x) = f(Mx+a) + b.x + c.
