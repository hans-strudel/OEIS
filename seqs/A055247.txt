%N A055247 Related to A055246 and A005836. Used for boundaries of open intervals which have to be erased in the Cantor middle third set construction.
%C A055247 At step number k >= 1 the 2^(k-1) open intervals which are erased from [0,1] in the Cantor middle third set construction are: I(k,n)=(a(2*n-1)/3^k,a(2*n)/3^k), n=1,.,2^(k-1).
%F A055247 a(n)= A055246((n+1)/2) if n odd, a(n)= A055246(n/2)+1 if n even. A055246(n)= 1+6*A005836(n-1), n >= 1.
%F A055247 a(1) = 1, a(n) = 3^ceiling( log_2( n ) ) - a( 2^ceiling( log_2( n ) ) + 1 - n ). - Juan Jose Alba Gonzalez (jjalba(AT)gmail.com), May 16 2006
