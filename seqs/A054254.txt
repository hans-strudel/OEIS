%N A054254 a(n) is n plus the minimum of the a(i)*a(n-i) of the previous i=1..n-1.
%C A054254 If in the Maple code "if n<=2 then n" was replaced by "if n<=1 then n", then the sequence would become the triangular numbers A000217. In general, if the Maple code was "if n<=k then n" for some given k >0 then a(n) would be n if n<=k, n+k(n-k) if k<=n<=2k and n(n+1)/ 2-k(k-1) if 2k<=n. - _Henry Bottomley_, Mar 30 2001
