%N A005344 a(n) = solution to the postage stamp problem with n denominations and 9 stamps.
%C A005344 _Fred Lunnon_ [W. F. Lunnon] defines "solution" to be the smallest value not obtainable by the best set of stamps. The solutions given are one lower than this, that is, the sequence gives the largest number obtainable without a break using the best set of stamps.
