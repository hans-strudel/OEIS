%N A005602 Smallest prime beginning a complete Cunningham chain of length n (of the first kind).
%C A005602 The word "complete" indicates each chain is exactly n primes long (i.e. the chain cannot be a subchain of another one). Except for a(1), each term, by definition, is a Sophie Germain prime (A005384) as is each element except the last in each chain; each element after the first in each chain is a safe prime (A005385), so interior elements are both.
