%F A053381 Let f(n) be the number of linearly independent smooth nowhere-zero vector fields on an n-sphere. Then f(n) = 2^c + 8d - 1 where n+1 = (2a+1) 2^b and b = c+4d and 0 <= c <= 3. f(n) = 0 if n is even.
